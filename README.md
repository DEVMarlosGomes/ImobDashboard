# 🏠 Imob Dashboard

Dashboard imobiliário interativo construído com HTML, CSS e JavaScript puro — sem frameworks,
sem dependências pesadas. Um projeto que une minha experiência como corretor de imóveis
com minha formação como desenvolvedor fullstack.

## ✦ Demo ao vivo

**[→ Ver dashboard](https://devmarlosgomes.github.io/ImobDashboard/)**

---

## 📊 Funcionalidades

- **KPIs em tempo real** — volume de vendas, imóveis fechados, leads ativos e taxa de conversão
- **Gráfico de vendas mensal** com 4 períodos alternáveis (7d, 30d, 90d, 12m)
- **Funil de vendas** com 5 estágios e animações de entrada
- **Mapa de leads** em SVG puro — sem APIs externas — com pins pulsantes por temperatura do lead
- **Gestão de leads** com status (quente, morno, frio, fechado)
- **Metas do mês** com barras de progresso e score geral
- **Feed de atividade** com histórico de ações recentes
- Saudação dinâmica baseada no horário real do usuário

---

## 🛠️ Stack

| Camada | Tecnologia |
|---|---|
| Estrutura | HTML5 semântico |
| Estilo | CSS3 com variáveis customizadas |
| Lógica | JavaScript ES6+ vanilla |
| Gráficos | Chart.js 4.4 |
| Tipografia | DM Serif Display + DM Sans |
| Deploy | GitHub Pages |

---

## 💡 Diferencial

Este projeto nasceu da combinação de dois mundos que domino:
como **corretor**, sei quais métricas realmente importam no dia a dia de vendas;
como **desenvolvedor**, sei como apresentá-las de forma clara e funcional.
O resultado é um dashboard que poderia ser usado por qualquer corretor ou imobiliária de verdade.
